Features


Blue Lotus brings many improvements to Minecraft's boring world.

  • 🎭 Play your way!
      • Start your game with different Origins or stick to the classic vanilla gameplay!

  • 🗺️ Improved world generation
      • Including better terrain with Tectonic v2, many new biomes added by Regions Unexplored & Nullscape, countless structure mods such as Towns and Towers , Repurposed Structures and Yung's Better Structures.

  • 🍃 Enhanced visuals and Immerison
      • Ambient Sounds, built in shader support with Iris, dynamic lighting, FogLooksGoodNow and some awsome particle improvements thanks to Explosive Enhancement and Falling Leaves.

  • 💎 New Materials
      • Many new, unique and interesting ores added by Mythic Metals

  • 🏠 Builders Rejoice!
      • Infinite building possibilities and over 9000 new blocks thanks to Chipped!

  • 🛒 Minecart improvements
      • Connectable carts with Linkart and High-Speed Rail, ice boats are no longer the fastest way to travel!

  • 🦴 Archeology on a new level!
      • Improvements thanks to Archeology Plus and Better Archeology

  • 🤠 Express yourself!
      • Fabric Tailor, Ears, Bedspreads, More Armor Trims, Elytra Trims and Emoji Type help you express yourself more than ever!

  • 📈 General QoL improvements
      • Supplementaries, Universal Enchants, Right Click Harvest, Reacharound, Leaves Be Gone, Better Statistics Screen...

  • And many, many more...

How to install?

For players:

Download a launcher that supports modrinth (e.g. Prism Launcher) then follow the installation instructions for the launcher of your choice.

For Servers:

  • Download the latest version of mrpack-install
  • Open the command line, navigate to the directory
  • Run mrpack-install bluelotus-beyond
  • Wait for the download to finish
  • Navigate into the new "mc" folder
  • Now run java -Xmx3G -Xms1G -jar quilt-server-launch.jar
      • Set -Xmx to the amount of RAM you want your server to use. (at least 2.5GB)
  • You're Done!

🔄 Verisoning :

The modpack uses a new version system: Major.Minor.Patch

  • Major : A new version of minecraft or a big modpack change, these updates are not backwards compatible.
  • Minor : New mods might have been added or removed, read the changelogs to see if it's backwards compatible.
  • Patch : Updated mods, safe to update worlds.
