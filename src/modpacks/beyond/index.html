How to install?

For Players

Download the Modrinth App or a launcher that supports modrinth (e.g. Prism Launcher) then follow the installation instructions for the launcher of your choice.

Modpack page on Modrinth

For Servers

  • Download the latest version of mrpack-install
  • Open the command line, navigate to the directory
  • Run mrpack-install bluelotus-beyond {{version.beyond}}
  • Wait for the download to finish
  • Navigate into the new "mc" folder
  • Now run java -Xmx3G -Xms1G -jar quilt-server-launch.jar
      • Set -Xmx to the amount of RAM you want your server to use. (at least 2.5GB)
  • You're Done!
